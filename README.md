#### Simple dice roller, WIP so I can teach myself JavaScript.

### Working Features

- Roll a d20: `!ico d20`
- Roll a high-variance d20: `!ico hv20`

### Planned Features

- Roll any number of sides: `!ico dx`
- Roll any number of high-variance sides: `!ico hvx`

*... where `x` is an integer.*
